### Leia os procedimentos para executar a Roleta laravel

Seu computador precisará ter o PHP com a versão >= 7.0.0 e suas extensões:

Você também vai precisar rodar o comando composer update (via CMD) na pasta raiz do projeto. Para isso terá que instalar o Composer caso ainda não tenha instalado.

Depois disso, basta rodar php artisan serve na raiz do projeto. Abra o nessa URL http://localhost:8080, e assim verá o projeto.

-----------------------------------------------
Caso não tenha o PHP instalado em sua máquina, siga esses procedimentos para executar o projeto:

Resumindo:

1 - Instalar o PHP e extensões do Laravel
2 - Instalar o Composer
3 - Baixar as dependências do projeto usando composer update
4 - Rodar php artisan serve
